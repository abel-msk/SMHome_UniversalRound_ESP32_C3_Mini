# SMHome_UniversalRound_ESP32_C3_Mini

This is small pcb board was designed to place  under wall mounted electric components in socket box, 
it steel can bee plcad in its own box  znd powered by usb.

## Haw to use

```
git clone https://github.com/abel-msk/SMHome_UniversalRound_ESP32_C3_Mini.git
cd SMHome_UniversalRound_ESP32_C3_Mini
```
Next step is to open project file  `SMHome_UniversalRound_ESP32_C3_Mini_v3.kicad_pro`  in KiKad.
